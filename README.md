# Интерактивные образовательные программы для интерактивных школьных досок по физике, электродинамике, стереометрии.

# Содержание:
1. обзор проекта
2. описание кода/архитектуры, навигация по коду

## 1 обзор проекта

Видеообзор:
https://rutube.ru/video/a8efeeca90e0aae2f7dd2995a7d57448/

Функционал:

- Воспроизведение аудио, видео, слайд шоу, анимаций содержащихся в образовательных интерактивных курсах
- Управление интерактивными симуляциями различных физических и электрических процессов
- Возможность использования курсов на интерактивной школьной доске
- Возможность использования курсов одновременно несколькими пользователями на одном компьютере


Задачи:

- Создание готового к продаже продукта. Включая написание скриптов для установщиков Windows (installShield Wizard)
- Добавление поддержки интерактивной школьной доски в ЦМС курсов
- Добавление поддержки нескольких мышей в ЦМС курсов
- Разработка интерактивных обучающих курсов по указаниям, макетам и формулам (для симуляций процессов в виртуальных лабораториях) методиста, анализирующего школьные учебники.


Инициативность:

- Существующие методы разработки, формат контента были усовершенствованы
- Были разработаны и успешно внедрены форматы новых более сложных видов интерактивного контента
- Существенный объем контента был преобразован в более простой и беспроблемный формат (ручная обработка контента постепенно была автоматизирована)
- Спроектирована и реализована, на более совершенной версии языка программирования, ЦМС  для интерактивных тестов
